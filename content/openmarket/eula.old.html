The OpenMarket Matrix API (the “OpenMarket Matrix API” or “API”) mentioned below refers to the interface hosted at matrix.openmarket.com enabling linking of messaging and voice calls happening over IP in the Matrix ecosystem with the Public Switched Telephone Network (PSTN). As such it allows sending an instant message from a Matrix client to a mobile phone who will receive it as an SMS and vice versa: send an SMS from a mobile phone to a virtual number linked to a Matrix user who will receive it as a Matrix instant message  in his Matrix client.
