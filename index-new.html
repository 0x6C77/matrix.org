Matrix is a new open standard for decentralised communication, providing simple HTTP APIs and open source reference implementations for securely distributing and persisting JSON over an open federation of servers.

Uses include:

  • Group Chat - fully distributed persistent chatrooms with no single points of control or failure
  • WebRTC Signalling - interoperable VoIP signalling for the web
  • Internet Of Things - exchanging and persisting data between devices and services
  • ...and just about any service where you need to pubsub persistent data.

Our mission is to let users own and control their own data and enable VoIP, IM and IoT services which are as universal and interoperable as email is today.

Open APIs

  • Simple pragmatic RESTful HTTP/JSON APIs
  • Open specification of the Matrix standard
  • Create and manage fully distributed (with eventual consistency) communication history no single points of control or failure
  • Send and receive extensible messages with optional end-to-end encryption
  • WebRTC VoIP/Video calling using Matrix signalling
  • Real-time synchronised history and state across all clients
  • Group communication by default
  • Use existing 3rd party IDs (e.g. email, phone numbers, Facebook) to authenticate, identify and discover end-users
  • Trusted federation of identity servers, tracking public keys and 3rd party ID mappings
  • TLS by default
