Seaglass is a Matrix client for macOS under active development. It is written in Swift, uses the [matrix-ios-sdk](https://github.com/matrix-org/matrix-ios-sdk) and the Cocoa user interface framework. It features working support for end-to-end encryption.

- Code and issues: https://github.com/neilalexander/seaglass
