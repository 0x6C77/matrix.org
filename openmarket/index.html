Experimental Two-way SMS for Matrix Users

OpenMarket Matrix Gateway

Home » Matrix Gateway

Overview

Matrix.org is a new open source project that defines standard HTTP APIs for decentralised communication. It aims to make interoperable Instant Messaging, VoIP and IoT communication as ubiquitous and flexible as email or the web by creating a new ecosystem of federated communication applications and services. As of February 2015, the Matrix reference implementations and standard are in early beta development, and are suitable for developers and power-users wanting to experiment with building Matrix-compatible clients, services and gateways between Matrix and existing communication products. For more information, please see the main Matrix.org website.

OpenMarket is proud to power the first commercial gateway available to the Matrix ecosystem: the OpenMarket Matrix Gateway. The Gateway gives Matrix users the ability to send and receive SMSes to normal phone users on the Public Switched Telephone Network (PSTN), massively extending the reach of the Matrix ecosystem to non-Matrix users. The OpenMarket Matrix Gateway may also be used as an easy way for Matrix developers to send and receive text messages using precisely the same APIs they use for sending and receiving Matrix instant messages today.

How to use the service

Sending SMS through the OpenMarket Matrix Gateway incurs no charge during our introductory beta testing period. In order to send messages we do require that you associate a valid PayPal account with the service for security purposes. Openmarket will not (and cannot) ever charge this account without your consent. Associating your PayPal account may be done via the settings page of any reference Matrix web client which has been configured to be aware of the OpenMarket Matrix Gateway. To use the matrix.org hosted web client, please visit https://matrix.org/beta/#/settings.

The settings page contains an "OpenMarket Matrix Gateway" section which guides you through the process of associating your PayPal account.

Having done this, you may start an SMS conversation with a user using the 'Start Chat' button in your preferred Matrix-compatible client application or website. PSTN users accessible via the OpenMarket Matrix Gateway have Matrix IDs of form @+<msisdn>:matrix.openmarket.com - e.g. @+447700900123:matrix.openmarket.com. Your client may give you a way to avoid manually entering the Matrix ID and allow you to directly talk to a given addressbook contact via SMS if that user is not on Matrix - this is up to the user interface of the client.

Whenever you send a SMS to a given user from Matrix for the first time, you are assigned a phone number from which the user will receive your message, which will allow them to respond to that message directly by SMS. Each user you converse with will receive a different number for contacting you. Each contact number will only work for that specific Matrix SMS conversation - your contacts should not share your contact number, as it will only work when they message it.. OpenMarket reserves the right to recycle contact numbers if they are not used for more than 3 months.

On sending the first SMS to a user, they will receive an "opt-in" message from the OpenMarket Matrix Gateway to invite them to join the conversation. This is equivalent to an invitation to join a room in Matrix. The user will have to respond to this message in order to receive your messages.

If you are an enterprise company requiring higher volume or a more flexible service, please contact us for more details.

Coverage

The OpenMarket Matrix Gateway is an experimental "labs" product. As of launch (February 2015) it supports SMS conversations only with users in the USA or United Kingdom.

Pricing

During beta testing, sending and receiving SMS incurs no cost - however it is subject to a per-user fair-usage policy.
